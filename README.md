### how to use
* enter any number you want, it will add to the current number counter
* add a minus sign (-) at the start to make the converted string subtract the current numbers
* add a slash (/) at the start to divide the current remaining integer (will crash if the number you divide by is 0)
* add an asterisk (*) at the start to multiply the the numbers
* add a percentage sign (%) to modulate the numbers that you currently hold

- thats it :>
